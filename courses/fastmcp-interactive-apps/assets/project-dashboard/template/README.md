# Project Dashboard

Starter project for the **Building Interactive Apps with FastMCP** track.

This project uses:

- `fastmcp[apps]==3.2.0`
- `prefab-ui==0.18.2`

Core files:

- `project_dashboard/server.py` - the MCP server you edit in each challenge
- `project_dashboard/helpers.py` - shared file/data helpers
- `project_dashboard/data/` - local JSON fixtures for the dashboard

Preview the app locally with:

```bash
source .venv/bin/activate
fastmcp dev apps project_dashboard/server.py
```
